# Zsh & Tmux Config

## Installation

```bash
git clone https://github.com/kara7z/zsh-config.git ~/zsh-config
cd ~/zsh-config
./setup.sh
```

## Tmux plugins

Open tmux and press `prefix + I` (Capital I) to install TPM plugins.

## Requirements

- [Oh My Zsh](https://ohmyz.sh/)
- [TPM](https://github.com/tmux-plugins/tpm) (installed automatically on first tmux launch)
